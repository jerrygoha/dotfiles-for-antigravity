# /pickup 가이드

> 이전 세션의 컨텍스트를 복원하는 워크플로우

---

## 📌 언제 사용하나요?

- `/handoff` 이후 새 세션 시작 시
- 다른 사람 작업 이어받을 때
- 휴식 후 복귀 시

---

## 💡 사용법

```
/pickup [파일명]
```

예: `/pickup 2026-01-21-oauth-implementation`

---

## 📤 복원 후 확인사항

1. **Pending Tasks** 먼저 확인
2. **Current Work** 섹션 파악
3. **Failed Approaches** 확인 (반복 방지!)
4. **Next Steps** 유효성 검토

---

## ✅ 검증 체크리스트

- [ ] 파일 경로가 여전히 존재하는가?
- [ ] 핸드오프 이후 다른 변경이 있었는가?
- [ ] Pending Tasks가 여전히 유효한가?
- [ ] 의존성이 변경되지 않았는가?

---

## 🔗 관련 워크플로우

- `/handoff` - 세션 컨텍스트 저장
- `/learn` - 패턴 추출
