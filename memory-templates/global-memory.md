# Global Memory Template

> Template for global memory configuration in Antigravity.

## Instructions

This template provides a starting point for your global memory. Copy and customize based on your needs.

---

```markdown
# [PROJECT CONTEXT]

## Active Projects
- **Project Name**: [description]
- **Tech Stack**: [languages, frameworks, tools]
- **Key Decisions**: [architecture decisions, patterns used]

## Preferences

### Code Style
- Language: [TypeScript/Python/Go/etc.]
- Formatting: [Prettier/Black/gofmt]
- Linting: [ESLint/Pylint/golangci-lint]

### Communication
- Primary Language: [English/Korean/etc.]
- Response Style: [Concise/Detailed]
- Code Comments: [Language preference]

## Recurring Tasks
- [Task 1]: [description]
- [Task 2]: [description]

## Important Links
- Repository: [URL]
- Documentation: [URL]
- CI/CD: [URL]
```

---

## Usage Tips

1. **Keep it focused**: Only include information that's relevant across sessions
2. **Update regularly**: Remove outdated information
3. **Be specific**: Vague context is less helpful than no context
