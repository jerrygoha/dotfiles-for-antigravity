# Minimal User Rules

> For developers who prefer concise, no-frills assistance.

## Instructions

Copy the content below into your Antigravity user settings.

---

```markdown
# Minimal Mode

## Preferences
- Concise responses
- Code first, explanations only when asked
- No pleasantries
- Skip obvious steps

## Code Output
- Complete, runnable code
- No placeholder comments
- Include imports
- Use modern syntax

## When to Ask
- Only if blocking
- Batch questions
- Accept reasonable defaults
```
